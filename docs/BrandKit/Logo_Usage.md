# Nice Place Logo Usage

## Official Logo Files

| File | Usage |
|---|---|
| `01_App_Icon.png` | App launcher icon, app store, favicon family |
| `02_Logo_Light.png` | Light backgrounds, documents, website light sections |
| `03_Logo_Dark.png` | Dark UI with green icon and white text |
| `04_Logo_Green.png` | Monochrome green usage, stickers, watermark |
| `05_Logo_Black.png` | Print, documents, black & white assets |
| `06_Logo_White.png` | Dark backgrounds, splash, footer, presentations |

## Logo Rules

Do not:
- Redesign the icon
- Change the number of pins
- Change route geometry
- Add gradients
- Add shadows
- Add glow
- Add 3D effects
- Stretch the logo
- Rotate the logo
- Put the logo in random containers
- Use unapproved colors

## Clear Space

Keep empty space around the logo equal to at least the height of the small tagline text.

## Minimum Size

Do not use the horizontal logo so small that the tagline becomes unreadable. If the tagline is unreadable, use only the app icon or a no-tagline logo variant in future.

## Preferred Splash Usage

For dark splash screens, use:
- Background: `#060A10`
- Logo: `03_Logo_Dark.png` or `06_Logo_White.png`
