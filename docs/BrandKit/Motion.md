# Nice Place Motion

Motion should feel premium, calm, and intentional.

## Timing

| Token | Value |
|---|---:|
| Fast | 150ms |
| Normal | 250ms |
| Slow | 400ms |

## Interaction Scale

| Token | Value |
|---|---:|
| Pressed | 0.96 |
| Active | 1.08 |

## Animation Rules

Use:
- Fade
- Scale
- Gentle slide
- Spring for bottom sheets and FAB-style actions

Avoid:
- Bouncy cartoon motion
- Spinning logo
- Glow animations
- Excessive Lottie effects
- Long delays

## Splash Direction

Preferred animated splash:
1. Native splash appears very briefly.
2. Animated Splash screen opens on `#060A10`.
3. Route/icon animation draws in.
4. `Nice Place` fades in.
5. Tagline fades in.
6. Entire splash fades out into the map.
