# Nice Place Brand Kit v1.0

This is the official visual identity package for Nice Place.

These assets are final. Do not redesign, replace, recolor, stretch, or recreate the logo unless a future brand revision is explicitly approved.

## Brand Direction

Nice Place is a premium, calm, outdoor, map-focused mobile app for discovering, saving, sharing, and remembering beautiful places.

Core feeling:
- Premium
- Calm
- Outdoor
- Travel
- Map-focused
- Modern
- Trustworthy
- Timeless

## Core Tokens

Primary Green: `#44A878`  
Accent Amber: `#E8A04A`  
Dark Background: `#060A10`  
Main Text Dark: `#1F2937`  
Main Text Light: `#FFFFFF`

Typography: Plus Jakarta Sans  
Icon System: Lucide
