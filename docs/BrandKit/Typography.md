# Nice Place Typography

Official font: **Plus Jakarta Sans**

## Weights

| Usage | Weight |
|---|---:|
| Brand name | Bold 700 |
| Page titles | Bold 700 |
| Section titles | SemiBold 600 |
| Card titles | SemiBold 600 |
| Body text | Medium 500 / Regular 400 |
| Captions | Medium 500 |
| Logo tagline | Medium 500 with letter spacing |

## Logo Text

Brand name:
```txt
Nice Place
```
Font: Plus Jakarta Sans Bold  
Tracking: 0

Tagline:
```txt
DISCOVER • SHARE • REMEMBER
```
Font: Plus Jakarta Sans Medium  
Letter spacing: approximately 3%

## Rule

Do not mix random fonts into the UI. Use Plus Jakarta Sans everywhere unless a system fallback is required temporarily.
